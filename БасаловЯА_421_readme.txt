https://github.com/Marilius/VKR

Требования: 
    python 3.12
    networkx
    metis

Для визуализации дополнительно:
    numpy
    matplotlib


Запуск программы (входные файлы заданы в методе research, который может потребоваться отредактировать):
python -m greed (должен быть выполнен до запуска остальных)
python -m GAP
python -m GAP2

Результаты для жадного алгоритма будут записаны в ./results/greed
Результаты для ГА_1-4 алгоритма будут записаны в ./results в папки с GAP
Результаты для ГА_1-6 алгоритма будут записаны в ./results в папки с GAP2
Результаты для кластерного алгоритма будут записаны в ./results2/MK_greed_greed_weighted
Результаты для кластерного алгоритма с ослаблением ограничения будут записаны в ./results2/MK_greed_greed_with_geq_cr


Построение графиков:
    В research.ipynb выполнить первую ячейку и следующие за заголовком "Построение диаграмм и тепловых карт для ВКР" ячейки, задав имена графов и параметры разбиений.
